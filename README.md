# Adversarial Attacks & Defense Mechanisms in Deep Learning Models

An interactive, high-fidelity research dashboard and production-ready PyTorch deep learning platform built to analyze, visualize, and defend against adversarial machine learning attacks (like FGSM and PGD).

This repository contains a **production-grade PyTorch deep learning modeling suite** in the `core/` folder (for neural network training, gradient-ascent attack optimization, and preprocessing filter convolutions) integrated directly with a **premium, zero-dependency visual research portal** to simulate and evaluate these security threats in real-time.

---

##  Key Features

*   **Dual-Attack Implementation:** Mathematical implementations of the **Fast Gradient Sign Method (FGSM)** and iterative **Projected Gradient Descent (PGD)** attacks w.r.t input pixels in PyTorch.
*   **Multi-Tier Defense Sandbox:**
    *   *Model Weight Regularization:* **Adversarial Training** utilizing online minimax gradient-ascent optimization.
    *   *Input Preprocessing Filters:* Average Spatial Smoothing, JPEG Compression simulation, and Bit-depth Quantization.
*   **Premium Interactive Visual Workspace:** A slate-dark glassmorphic web dashboard containing:
    *   *Live Attack Sandbox:* Adjust an $\epsilon$-perturbation slider in real-time to convolve noise masks and witness model predictions shift dynamically on HTML canvases.
    *   *Responsive SVG Chart Engine:* Renders custom coordinate curves (Accuracy vs. Epsilon decay) mapping standard models, robust models, and preprocessors side-by-side.
    *   *Terminal Log Typing Simulator:* Typographical simulation of PyTorch backend training epoch execution feeds.
    *   *Educational Lab:* Visual breakdowns of backpropagation vectors, math formulas, and copy-pasteable PyTorch code.
*   **Dual Data Synchronization:** The web dashboard searches for `results.json` generated by the backend suite to render your live computed weights, falling back to a pre-compiled high-fidelity database if launched offline via `file://`.

##  Getting Started

To launch and run this project locally, you can preview the visual dashboard and execute the PyTorch deep learning mathematical models.

### Part 1: Running the Visual Dashboard on Localhost

Running on `localhost` bypassing browser security restrictions w.r.t local files, allowing the dashboard to read the computed `results.json` directly from the filesystem.

Since **Python 3.11** is already installed on your system, this requires zero setup:

1.  Open your **PowerShell** or **Command Prompt** and navigate to your project directory:
    ```powershell
    cd "d:\Sixth sem\AML\project"
    ```
2.  Start the lightweight Python web server:
    ```powershell
    python -m http.server 8000
    ```
3.  Open your browser and navigate to:
    ### **[http://localhost:8000](http://localhost:8000)**

---

### Part 2: Executing the PyTorch Machine Learning Core

To retrain the models, execute the gradient attacks, and refresh the visual assets:

1.  Activate your virtual environment and install standard requirements:
    ```powershell
    # Windows PowerShell Activation
    python -m venv venv
    .\venv\Scripts\Activate.ps1

    # Install packages
    pip install -r core/requirements.txt
    ```
2.  Execute the master pipeline runner:
    ```powershell
    python core/run_suite.py
    ```
3.  The runner trains standard and robust CNN models on MNIST for 5 epochs, benchmarks accuracy across epsilons, saves visual matrices in `assets/images/`, and overwrites `results.json`. The localhost webpage will immediately refresh to display your new live computed curves!

*   **Standard CNN Baseline Clean Test Accuracy:** **`98.7%`** (Fragile: decays to **`11.2%`** under FGSM @ $\epsilon=0.15$)
