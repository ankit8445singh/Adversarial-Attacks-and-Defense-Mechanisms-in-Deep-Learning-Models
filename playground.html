1. Attack Parameters

PERTURBATION INTENSITY BOUNDARY (ε) 0.05
ε = 0.00 (Clean) ε = 0.30 (Strong Perturbation)
FGSM (Fast Gradient Sign Method) Single-step gradient update to maximize loss: x + ε * sign(grad)
PGD (Projected Gradient Descent) Iterative optimization step projected back inside epsilon-ball

2. Defense Strategy

No Defense (Standard Model Vulnerability) Exposes model directly to raw perturbed pixel streams.
Input Preprocessing: Spatial Avg Smoothing Applies low-pass 3x3 pooling to blur out adversarial high frequencies.
Input Preprocessing: JPEG Compression Simulates 8x8 macroblock quantization to drop fine gradient maps.
Input Preprocessing: Bit-Depth Quantization Reduces precision to 3 bits, compressing imperceptible noises.
Robust Model: Online Adversarial Training Switches weights to model trained with online FGSM inputs.
